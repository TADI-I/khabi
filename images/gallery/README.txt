HOW TO ADD PHOTOS TO THE GALLERY
=================================

1. Place your image files in this folder: images/gallery/

2. Name them to match the gallery categories:

   Construction photos:
     construction-1.jpg
     construction-2.jpg
     construction-3.jpg
     construction-4.jpg
     (add more: construction-5.jpg, etc.)

   Cleaning photos:
     cleaning-1.jpg
     cleaning-2.jpg
     cleaning-3.jpg

   Renovation photos:
     renovation-1.jpg
     renovation-2.jpg

3. Recommended image size: 1200x900px or similar 4:3 ratio
   Formats supported: .jpg, .jpeg, .png, .webp

4. To add MORE gallery items, open index.html and copy one of the
   <div class="gallery-item"> blocks inside the galleryGrid section,
   changing the image src and alt text.
